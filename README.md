# Psychic-Volume

🎚️ Adjust your MacBook volume with just a **pinch of your fingers** using computer vision!

---

## 📦 Built With
- **Python**
- **OpenCV**
- **MediaPipe**
- **NumPy**

---

## 🚀 Installation

Make sure you have Python 3 installed, then run the following commands:

```bash
pip install opencv-python
pip install mediapipe
pip install numpy
```
